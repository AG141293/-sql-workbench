# ⚡ SQL Workbench

A fully browser-based SQL editor — no backend, no installation required.

## 🌐 Live Demo
👉 [Open SQL Workbench](https://YOUR-USERNAME.github.io/sql-workbench/)

## ✨ Features
- Write and run SQL queries directly in the browser
- Built-in sample datasets (Employees, Sales, Products, Movies)
- Upload your own CSV files
- Schema viewer, query results table, and message log
- Keyboard shortcut: `Ctrl+Enter` to run queries
- Resizable editor and results panes

## 🛠 Tech Stack
- Vanilla HTML, CSS, JavaScript
- [SQL.js](https://sql.js.org/) — SQLite compiled to WebAssembly
- JetBrains Mono & Syne fonts (Google Fonts)

## 🚀 How to Use
1. Visit the live demo link above, OR
2. Clone this repo and open `index.html` in any browser
